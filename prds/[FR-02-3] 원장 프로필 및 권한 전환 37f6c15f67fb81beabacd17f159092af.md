## 1. 개요

### 목적

- 원장 권한 보유 사용자가 보호자 화면과 원장 화면을 명확하게 전환할 수 있게 한다.
- 원장 화면에서 원생 보호자에게 노출되는 원장 프로필을 관리할 수 있게 한다.

### 배경

- v3.0에서는 같은 계정이 보호자 기능과 원장 기능을 함께 사용할 수 있으므로, 현재 사용 중인 역할 화면을 명확히 구분해야 한다.
- 권한 획득, 권한 해제, 계정 탈퇴, 이미지 정책은 별도 문서에서 관리하고, 본 문서는 화면 전환과 원장 프로필 관리에 집중한다.

### 대상 사용자

| 사용자 | 상태 | 목표 |
| --- | --- | --- |
| 원장 | 유치원 관리 권한 보유 | 보호자 화면과 원장 화면을 전환하고 원장 프로필을 관리한다. |

### 성공 기준

- 원장 권한 보유 사용자는 보호자 마이페이지에서 `원장으로 전환`을 눌러 원장 홈으로 이동할 수 있다.
    - 원장 화면 하단 탭은 `홈`, `알림장`, `구성원`, `마이`로 구성된다.
- 원장 마이페이지에서 `보호자로 전환`을 누르면 보호자 화면의 `내 주변`으로 이동한다.
    - 보호자 화면 하단 탭은 `내 주변`, `보관함`, `유치원`, `마이`로 구성된다.
- 원장 프로필에서 이름, 연락처, 이메일을 표시/수정할 수 있다.

## 5. 정책

| 정책 | 내용 |
| --- | --- |
| 전환 버튼 노출 | 보호자 마이페이지 상단 `원장으로 전환`은 원장 권한을 보유한 사용자에게만 노출한다. 권한이 없으면 버튼을 노출하지 않는다. |
| 전환 후 진입 화면 | 보호자에서 원장으로 전환하면 원장 홈으로 이동한다. 원장에서 보호자로 전환하면 내 주변으로 이동한다. |
| 원장 권한 해제 메뉴 | 원장 마이페이지 `내 계정` 아래 `원장 권한 해제하기` 메뉴는 노출만 본 문서에서 정의한다. 상세 해제 정책은 [FR-02-5]를 따른다. |
| 원장 프로필 초기값 | 원장 인증 절차에서 입력한 이름과 연락처를 원장 프로필 초기값으로 사용한다. |
| 원장 프로필 수정 | 원장은 이름, 연락처, 이메일을 수정할 수 있다. 이름은 실명과 다르게 표시명으로 설정할 수 있다. |
| 원장 개인 주소 제외 | 원장 개인 주소는 원장 프로필에 표시하거나 수정 항목으로 제공하지 않는다. |
| 프로필 이미지 | 원장 프로필 이미지는 기본 없음 상태로 시작한다. 업로드/수정/삭제 정책은 공통 프로필 이미지 정책을 따른다. |
